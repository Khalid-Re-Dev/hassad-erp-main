## Developer Notes

### Refactoring Pattern Applied:

```python
# 1. Import modern components
from ui.layout_components import Card, Toolbar, FilterBar, DataHeader, FormSection, Spacing
from ui.animations import fade_in, sequential_card_reveal, AnimationDuration

# 2. Replace flat layouts with Cards
main_card = Card()
section_card = Card("Title | العنوان", collapsible=True)

# 3. Use FormSection for 2-column forms
form = FormSection(columns=2)
form.add_field("Label | التسمية", widget)

# 4. Add Toolbar and FilterBar
toolbar = Toolbar()
toolbar.add_action("Action | إجراء", callback, primary=True)

filter_bar = FilterBar("Search... | بحث...")
filter_bar.add_filter("Filter | فلتر", options, callback)

# 5. Apply animations
QTimer.singleShot(50, lambda: fade_in(main_card))
```

### Common Patterns:
- **Search handling**: `filter_bar.search_changed.connect(self._on_search)`
- **Filter handling**: `filter_bar.add_filter(label, options, self._on_filter)`
- **Header updates**: `self.header.set_count(count, "items | عناصر")`
- **Bilingual format**: `"English | العربية"`
