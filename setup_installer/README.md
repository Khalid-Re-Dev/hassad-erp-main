# Setup Installer (Future Phase)

This directory will contain installation packages and setup scripts for distributing the Hassad ERP System as a desktop application.

## Planned Features

- Windows installer (.exe)
- macOS installer (.dmg)
- Linux packages (.deb, .rpm)
- Automatic PostgreSQL setup
- Database initialization
- Desktop shortcuts
- Auto-update mechanism

## Implementation

This will be implemented in Phase 6 after all core features are complete.

## Technologies

- PyInstaller or cx_Freeze for Python packaging
- Inno Setup for Windows installer
- DMG Canvas for macOS installer
- FPM for Linux packages
